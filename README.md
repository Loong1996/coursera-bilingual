# Coursera 多语字幕

在 Coursera 视频中同时叠加显示多种语言字幕的 Chrome 扩展，支持自定义语言组合与显示顺序。

## 功能

- **自动开启**：进入 Coursera 视频页面时自动激活所选语言字幕
- **多语同显**：可同时叠加任意数量的字幕语言，不限于两种
- **固定行序**：各语言字幕按设定顺序从上到下排列，不再随机
- **自定义语言**：从视频实际提供的字幕轨道中勾选任意语言组合
- **调整顺序**：通过 ▲▼ 按钮调整各语言的显示行序
- **持久保存**：语言偏好保存到 Chrome 同步存储，换视频后自动沿用
- **SPA 兼容**：Coursera 单页应用内跳转视频时自动重新应用

## 安装

1. 下载或克隆本仓库到本地
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启**开发者模式**
4. 点击**加载已解压的扩展程序**，选择本项目文件夹

## 使用

打开任意 Coursera 视频页面，扩展会自动检测并启用字幕。

点击工具栏中的扩展图标可以：

- 勾选/取消勾选要显示的字幕语言（可选多种）
- 用 ▲▼ 按钮调整各语言的上下显示顺序
- 点击**手动重新应用**在自动检测失败时强制触发

默认显示英文（第一行）+ 中文简体（第二行）。若视频提供更多语言，可额外勾选叠加为第三行、第四行……

## 示例

| 勾选语言 | 显示效果 |
|---|---|
| English → 中文（简体） | 英文在上，中文在下（双语） |
| English → 中文（简体） → 日本語 | 英文 / 中文 / 日文三行叠加 |
| 中文（简体） → English | 中文在上，英文在下 |

## 文件结构

```
├── manifest.json   # 扩展配置（Manifest V3）
├── content.js      # 注入页面的核心脚本
├── popup.html      # 点击图标时的弹窗界面
├── popup.js        # 弹窗交互逻辑
└── icons/          # 扩展图标
```
